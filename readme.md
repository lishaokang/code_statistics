# 代码分析脚本

####  运行环境：
 - python
 - xlwt库 `pip install xlwt`

#### 添加项目名称
```text
1. 项目负责人员需要将项目名称添加到main.py代码的projectsList列表中
2. 代码存放路径需要修改baseDir的值，默认为当前目录，此处注意：程序默认会强拉取更新。所以如果暂存区有修改请单独创建统计代码目录并clone代码。
```
#### run python
```text
python main.py
执行完成后会在当前目录下生成统计结果表格code_statistics.xlsx
```
